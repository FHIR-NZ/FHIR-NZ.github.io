Namespaces

The standards.digital.health.nz domain is used for allocating identifiers for naming systems, code systems, terminologies, and other namespaces for use in the NZ digital health ecosystem.

Each namespace is identified by a Uniform Resource Identifier (URI) which will resolve to a page containing further information about that namespace.

Below are the namespaces allocated so far.


Identifier Namespaces

NHI Number: https://standards.digital.health.nz/id/nhi
HPI Person Identifier: https://standards.digital.health.nz/id/hpi-person
HPI Organisation Identifier: https://standards.digital.health.nz/id/hpi-organisation
HPI Facility Identifier: https://standards.digital.health.nz/id/hpi-facility
