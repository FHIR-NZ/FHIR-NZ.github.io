Namespaces

The standards.digital.health.nz domain is used for allocating namespaces for identifiers, code systems, terminologies, and value sets for use in the NZ digital health ecosystem.

Each namespace is identified by a Uniform Resource Identifier (URI) which will resolve to a page containing further information about that namespace.
